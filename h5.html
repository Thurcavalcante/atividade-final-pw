HTML

O que é HTML?

HTML, significa Hyper Text Markup Language, "linguagem de marcação de hiper texto". Marcação padrão para a criação de páginas da web. Descreve a estrutura de uma página da web que consiste em uma série de elementos HTML, esses elementos informam ao navegador como exibir o conteúdo. Os elementos identificam partes de conteúdo como "isto é um título", "isto é um parágrafo", "isto é um link", etc.

Mas que é um elemento HTML?

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final: < tagname > O conteúdo vai aqui ... < / tagname >. O elemento HTML é tudo, desde a tag inicial até a tag final: < h1 > Meu primeiro título < / h1 > < p > Meu primeiro parágrafo. < / p >.

Criando a página

  • A < !DOCTYPE html > declaração define que este documento é um documento HTML5
  • O < html > elemento é o elemento raiz de uma página HTML
  • O < head > elemento contém metainformações sobre a página HTML
  • O < title > elemento especifica um título para a página HTML
    (que é mostrado na barra de título do navegador ou na guia da página)
  • O < body > elemento define o corpo do documento e é um contêiner para todo o conteúdo
    visível, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
  • O < h1 > elemento define um grande título
  • O < p > elemento define um parágrafo

Estrutura da Página HTML

Sintaxe

A marcação é feita através das tags; Tags são palavras chaves cercadas pelos sinais < > Ex.: < html >. Normalmente vêm em pares como < b > e < /b > Representam Início e Fim respectivamente; Alguns elementos HTML têm conteúdo vazio (ou seja, não tem a tag de fim e são fechados na própria tag de início) Exemplos: < br /> e < hr />.

Títulos de Seleção

Títulos HTML são títulos ou subtítulos que você deseja exibir em uma página da web. Os elementos do HTML, representam seis níveis de títulos de seção. Começando de < h1 > a < h6 >. Sendo < h1 > o nível de seção mais alto e < h6 > o nível mais baixo.


< h1 >Exemplo 1< /h1 >
< h2 >Exemplo 2< /h2 >
< h3 >Exemplo 3< /h3 >
< h4 >Exemplo 4< /h4 >
< h5 >Exemplo 5< /h5 >
< h6 >Exemplo 6< /h6 >

Títulos são importantes, os mecanismos de pesquisa usam os títulos para indexar a estrutura e o conteúdo de suas páginas da web. Os usuários costumam folhear uma página por seus títulos. É importante usar cabeçalhos para mostrar a estrutura do documento. < h1 >os cabeçalhos devem ser usados ​​para os cabeçalhos principais, seguidos dos < h2 > cabeçalhos, depois os menos importantes < h3 >e assim por diante.

Como se cria títulos de seção na página?

Então, basta digitar no seu editor de codigos um dos seis tipos de títulos de seção que vai lher servir melhor. É importante lembrar que, quando for criar o título de seção, colocar ele dentro da tag "body" Por exemplo: Para usar o título "h1", é como a maioria das outras tags em HTML. Ou seja, existe a tag de abertura < h1 > e a tag de fechamento < /h1 >, e o título é o texto que você vai colocar entre essas tags. Essas tags devem estar aninhadas dentro das tags < body > e < /body >.

Parágrafos

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto. O < p > elemento HTML define um parágrafo. Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

Exemplo:
< p >Este é um parágrafo< /p >
< p >Este também é um parágrafo< /p >.

Quebras de linhas

A tag < br > define uma quebra de linha. Use < br >se quiser uma quebra de linha (uma nova linha) sem iniciar um novo parágrafo:

Exemplo:
< p >Este é um parágrafo
< br >com quebra de linha< /p >.

Links HTML - Hiperlinks

Os links HTML são hiperlinks. Você pode clicar em um link e pular para outro documento. Quando você move o mouse sobre um link, a seta do mouse se transforma em uma pequena mão.

A tag < a > define um hiperlink. O atributo mais importante do < a > elemento é o "href" atributo, que indica o destino do link. O texto do link é a parte que ficará visível ao leitor. Clicar no texto do link enviará o leitor para o endereço URL especificado.

Exemplo:
< a href="https://www.google.com/">Clique aqui para ir para o Google< /a >

Tabelas

As tabelas permitem que os desenvolvedores da web organizem os dados em linhas e colunas. A tag < table > define uma tabela.

Cada linha da tabela é definida com uma < tr > tag. Cada cabeçalho da tabela é definido com uma < th > tag. Cada dado / célula da tabela é definido com uma < td > tag.

Por padrão, o texto nos < th > elementos está em negrito e centralizado. Por padrão, o texto nos < td > elementos é regular e alinhado à esquerda. Exemplo a baixo:


< table >
< tr >
< th >Nome< /th >
< th >Sobrenome< /th >
< th >Idade< /th >
< /tr >
< tr >
< td >João< /td >
< td >Gomes< /td >
< td >18 anos< /td >
< /tr >
< tr >
< td >Caetano< /td >
< td >Veloso< /td >
< td >78 anos< /td >
< /tr >
< /table >
